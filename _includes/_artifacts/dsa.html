For this enhancement, I'm using the same artifact which is the Inventory Manager Android application developed in CS-360 Mobile Programming between April and June 2024. The application includes a low-stock notification system, which was initially implemented using a linear search algorithm with O(n) complexity. This approach required scanning the entire dataset sequentially, making it increasingly inefficient as the dataset grew. While functional for small product catalogs, this method did not scale well for larger datasets and resulted in performance bottlenecks as the number of inventory items increased. I selected this artifact because it provides an ideal opportunity to showcase my ability to analyze and implement efficient data structures for real-world performance optimization.

To improve system performance and scalability, I integrated an AVL Tree, a self-balancing binary search tree that ensures efficient lookup, insertion, and deletion operations by maintaining a time complexity of O(log n). Unlike a standard BST, which can become unbalanced and degrade to O(n) search time in the worst case, the AVL Tree automatically balances itself through rotations which ensures consistently optimal search performance. This enhancement allows the system to remain responsive and scalable, regardless of inventory size. Moreover, it showcases my ability to evaluate algorithm trade-offs and optimize search operations for real-world applications. Additionally, to illustrate the benefits of the AVL Tree, I provided a side-by-side comparison of the linear search and AVL search algorithm tasks execution times. As shown, the AVL Tree consistently performed faster, demonstrating the real-world performance benefits of the enhancement.

This enhancement provides significant business benefits particularly for companies with large inventories. By improving the efficiency of low-stock alerts, employees can quickly identify inventory shortages, allowing for faster restocking or reordering. The reduced computational overhead ensures that as the inventory grows, the system remains highly responsive, preventing performance degradation over time. For businesses managing thousands of inventory items, this enhancement translates to increased operational efficiency, ensuring that stock levels are monitored in real-time without delays.
