The final artifact is a full-stack Inventory Management System evolved from a mobile-only Android application into a scalable, enterprise-ready web solution. Each enhancement contributes to creating a robust, secure, and efficient system designed to support businesses managing large-scale inventory operations. Migrating from Android to .NET 8 Blazor Server allows for cross-platform accessibility enabling users to manage inventory from any device. The integration of Microsoft SQL Server enhances data integrity, availability, and multi-user support, ensuring secure and concurrent inventory management. Additionally, optimizing low-stock notifications with an AVL Tree improves system performance, making it efficient even with growing datasets. These enhancements unify to create a scalable, enterprise-ready inventory management system, supporting multi-user access, efficient inventory processing, and secure authentication.

Features and Functionality:

  • Role-Based Authentication & Authorization - Implemented using Identity Framework, ensuring that only authorized userscan manage inventory.
  • User Management - Administrators can add, edit, and remove users, assigning roles to enforce access control.
  • Inventory Tracking - Users can add, edit, and delete inventory items, updating stock levels in real time.
  • Low-Stock Notifications - A self-balancing AVL Tree algorithm optimizes low-stock alerts, ensuring efficient lookup andprocessing of notifications.
  • Enterprise-Ready Database - Migrated from SQLite to Microsoft SQL Server, enabling multi-user access, data integrity,and secure remote access.
  • Cross-Platform Accessibility - With the migration to Blazor Server, the system is no longer restricted to Androiddevices, allowing access from desktops, tablets, and mobile browsers.
